Running Cron in Windows
20 November 2013

We all would have heard of Cron jobs, if you are involved in website management, development or working on Linux/Unix machines. What’s a cron? Cron is nothing but an demon job running in the background without any user intervention. It gets executed automatically based on the predefined time interval. Generally cron jobs are used to perform some maintenance activities in the server.
